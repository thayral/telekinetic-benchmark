# telekinetic-benchmark
(This work is under development and inspired by the DeepMind/Kaggle competition on cognitive benchmarks)

Trying to designed cognitive benchmarks for physical agents.
It is set in a "telekinesis" framework: agents control objects in an abstract way. The low-level complexity of physical interaction and dynamics is partly removed.
However, a lot of physical constraints remain: object collision, 3D space,...
Telekinesis corresponds to the object-centric level, which seems to be a good compromise for the cognitive evaluation of arbitrary embodiements in physical tasks.
The interface and evaluation protocols remain a big challenge.
